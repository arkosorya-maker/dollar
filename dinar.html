DinarAPI – Currency and Commodity Prices API

DinarAPI is the first currency and commodity prices API that provides the latest currency and commodity prices from the local markets in Kurdistan and Iraq. Using the API is easy: make a request and get the currency and commodity prices. Your project can use the currency and commodity prices from the DinarAPI website with ease and convenience.
